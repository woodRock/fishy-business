Installation
============

Prerequisites
-------------
- Python 3.9 or higher
- Graphviz (for evolutionary model visualizations)

Install from Source
-------------------

1. Clone the repository:

.. code-block:: bash

   git clone https://github.com/woodRock/fishy-business.git
   cd fishy-business

2. Install dependencies:

.. code-block:: bash

   pip install -r requirements.txt

3. (Optional) Install development dependencies for testing:

.. code-block:: bash

   pip install pytest pytest-cov pytest-mock
