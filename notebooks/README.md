# Tutorials and Examples (Thesis Contributions)

This directory contains Jupyter notebooks that demonstrate the full machine learning pipeline used in each contribution chapter of the doctoral thesis.

## Available Notebooks

1. **[01_Datasets_and_Preprocessing.ipynb](01_Datasets_and_Preprocessing.ipynb)**: Exploratory data analysis, filtering rules, and spectral visualizations (Mean Signatures, PCA, t-SNE).
2. **[02_Species_and_Part_Identification.ipynb](02_Species_and_Part_Identification.ipynb)**: Identification of fish species and anatomical parts using Transformers and Random Forests.
3. **[03_Oil_and_CrossSpecies_Adulteration.ipynb](03_Oil_and_CrossSpecies_Adulteration.ipynb)**: Detecting cross-species mixtures and mineral oil adulteration, including ordinal performance analysis.
4. **[04_Contrastive_Learning_for_Batch_Detection.ipynb](04_Contrastive_Learning_for_Batch_Detection.ipynb)**: Using self-supervised contrastive learning (SimCLR) for robust representation learning and batch anomaly detection.

## How to use

You can run these notebooks locally using Jupyter Lab or Jupyter Notebook:

```bash
pip install jupyterlab
jupyter lab
```

These notebooks are also integrated into the [official documentation](https://fishy-business.readthedocs.io/).
