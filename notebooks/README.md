# Tutorials and Examples

This directory contains Jupyter notebooks demonstrating the usage of the `fishy-business` library.

## Available Notebooks

1. **[01_getting_started.ipynb](01_getting_started.ipynb)**: Learn how to run a minimal training experiment using the high-level API.

## How to use

You can run these notebooks locally using Jupyter Lab or Jupyter Notebook:

```bash
pip install jupyterlab
jupyter lab
```

These notebooks are also integrated into the [official documentation](https://fishy-business.readthedocs.io/).
